## Claude Code vs Cursor 비교

### 핵심 차이점

| 측면 | Claude Code | Cursor |
|------|-------------|-----------|
| **인터페이스** | CLI (터미널 기반) | GUI (IDE 기반, VS Code 포크) |
| **컨텍스트** | 대규모 (200k 토큰) | 중소 규모 |
| **자율성** | 높음 (서브-에이전트 협업 가능) | 중간 (에이전트 모드, YOLO 모드) |
| **테스트/커밋** | 자동화 우수 | 수동 중심 |
| **비용** | 클라우드 유료 ($20~100/월), 로컬 Ollama 무료화 가능 | 무료 플랜 있음 ($20~200/월) |
| **팀 워크플로** | 승인 기반 (체크포인트 롤백) | 프로젝트 규칙 기반 |

**요약**: Claude Code는 "자율적 CEO 대리", Cursor는 "실시간 페어 프로그래밍"
