# 02. 시스템 사양 및 요구사항

성공적인 로컬 LLM 및 AutoGen 시스템 구축을 위한 최적의 하드웨어 사양과 환경 요구사항을 안내합니다.

## 1. 하드웨어 사양 (Mac M4 Pro 최적화)

본 가이드는 아래 사양을 기준으로 최적화되었습니다.

- **칩셋**: Apple M4 Pro (14코어 CPU, 20코어 GPU, 16코어 Neural Engine)
- **메모리**: **64GB 통합 메모리 (Unified Memory)**
    - *이유*: 대형 모델(Architect, Coder, Reviewer)을 동시에 메모리에 상주시키거나 빠른 교체를 위해 최소 64GB가 권장됩니다.
- **스토리지**: 1TB SSD (모델 파일 및 Docker 이미지 저장을 위해 충분한 공간 확보 필수)
- **네트워크**: 10 기가비트 이더넷 및 Thunderbolt 5 지원 (고속 데이터 전송 및 분산 환경 대비)

## 2. 모델 실행 능력

Mac M4 Pro 64GB 환경은 다음과 같은 모델 실행 능력을 갖추고 있습니다.

- **멀티 에이전트 동시 구동**: Architect, Coder, Reviewer 모델을 동시에 로드하여 지연 시간 없는 협업 가능.
- **추론 속도**: 4-bit 양자화 모델 기준 초당 10~20 토큰 이상의 안정적인 추론 속도 제공.
- **지연 로딩 및 메모리 반납**: 필요에 따라 사용하지 않는 모델의 메모리를 즉시 반납하여 리소스를 최적화.

## 3. 소프트웨어 요구사항

시스템 구축을 위해 다음 도구들이 설치되어야 합니다.

- **OS**: macOS 최신 버전 (M4 Pro 성능 최적화 반영)
- **런타임**: Python 3.10 이상, Node.js (OpenWebUI 등 활용 시)
- **가상화**: Docker Desktop for Mac
- **LLM 엔진**: Ollama
- **프레임워크**: AutoGen 0.4.x 이상

## 4. 핵심 환경 설정 전략

- **TDD (Test Driven Development)**: 개발 에이전트가 코드를 작성할 때 항상 테스트 코드를 동반하도록 구성.
- **메모리 관리**: Ollama의 지연 로딩 설정을 조정하여 불필요한 VRAM 점유 방지.
- **컨텍스트 공유**: 에이전트 간 대화 기록을 효율적으로 요약 및 공유하여 토큰 소모 최적화.
