# 시스템 사양 및 활용

### 당신의 Mac Mini M4 Pro 스펙
```
- Apple M4 Pro 칩 (14코어 CPU, 20코어 GPU, 16코어 Neural Engine)
- 64GB 통합 메모리 (최강 구성)
- 10 기가비트 이더넷
- 1TB SSD 저장 장치
- 3x Thunderbolt 5 포트
```

### 활용 능력
- **70B 모델** 4-bit 양자화 실행 가능 (10-20토큰/초)
- **Docker 컨테이너 다중 실행** 여유로움
- **10GbE + Thunderbolt 5**: 네트워크/외장 스토리지 초고속
- **동시 3개 LLM 실행 가능** (예: Scout 17GB + Gemma3 12B + DeepSeek 16B = 45GB)
