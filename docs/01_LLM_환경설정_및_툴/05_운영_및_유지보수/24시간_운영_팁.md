## 24시간 운영 팁

### 1. 전원 및 열 관리

#### 전원 설정
```bash
# 디스플레이 끄기 (맥은 켜두기)
시스템 설정 → 화면 시간 초과 → 디스플레이 끄기: 15분
시스템 설정 → 전원 → "컴퓨터가 절대 자도록 허용" 체크
```

#### 통풍
- 벽과 최소 10cm 간격
- 위/아래 공간 확보
- 정기적 통풍구 청소

#### 자동 재부팅 (주 1회 권장)
```bash
시스템 설정 → 소프트웨어 업데이트 → 자동 업데이트/재부팅 설정
또는 cron: 매주 일요일 03:00에 재부팅
```

### 3. 모니터링 및 로깅

#### 시스템 상태 확인
```bash
# CPU/메모리 실시간 모니터링
top -u

# GPU 사용률 (Apple Silicon)
powermetrics --samplers gpu_power -n 1

# 디스크 용량
df -h

# 온도 확인 (istats 설치 필요)
brew install istats
istats
```

#### 로그 저장
```bash
# Ollama 로그
log stream --predicate 'process == "ollama"' --level debug

# Docker 로그
docker logs -f open-webui

# AutoGen 스크립트 로그
tail -f /tmp/autogen.log
```

### 4. 정기 유지보수

```bash
# 주 1회
docker system prune  # 미사용 이미지/볼륨 정리
brew update && brew upgrade

# 월 1회
git maintenance run --auto  # Git repo 최적화
du -sh ~  # 디스크 사용량 체크

# 분기 1회
SSD TRIM 최적화 (자동으로 됨)
백업 확인
```

### 5. 자동 재시작 (에러 대비)

#### Ollama 재시작
```bash
brew services restart ollama
```

#### OpenWebUI Docker 재시작
```bash
docker restart open-webui
```

#### AutoGen 스크립트 재시작 (launchd)
```bash
launchctl stop com.local.autogen
launchctl start com.local.autogen
```
