## 로컬 코딩 어시스턴트 설정

### 옵션 1: Continue.dev (Cursor-like, VS Code 확장)

#### 설치
1. VS Code 열기
2. 확장 마켓플레이스에서 "Continue" 검색 설치
3. Ctrl+L (또는 Cmd+L) 단축키 확인

#### Ollama 연동 설정
```bash
# 설정 파일 위치
~/.continue/config.json

# 내용 (새로 생성)
cat > ~/.continue/config.json << 'EOF'
{
  "models": [
    {
      "title": "Llama 4 Scout",
      "provider": "ollama",
      "model": "llama4:scout-q4_K_M",
      "apiBase": "http://localhost:11434"
    },
    {
      "title": "DeepSeek-Coder 16B",
      "provider": "ollama",
      "model": "deepseek-coder-v2:16b",
      "apiBase": "http://localhost:11434"
    }
  ],
  "tabAutocompleteModel": {
    "title": "DeepSeek-Coder 16B",
    "provider": "ollama",
    "model": "deepseek-coder-v2:16b",
    "apiBase": "http://localhost:11434"
  }
}
EOF
```

#### 사용 방법
- `Ctrl+L` (또는 `Cmd+L`): 채팅 시작
- `Ctrl+K` (또는 `Cmd+K`): 코드 편집
- `Ctrl+Shift+L`: 파일 선택해서 컨텍스트 추가

### 옵션 2: Claude Code CLI (로컬 Ollama 연동)

#### Claude Code CLI 설치
```bash
curl -fsSL https://claude.ai/install.sh | bash
claude --version
```

#### Ollama와 연동 (완전 로컬화)
```bash
echo 'export ANTHROPIC_BASE_URL=http://localhost:11434' >> ~/.zshrc
echo 'export ANTHROPIC_API_KEY=ollama' >> ~/.zshrc
source ~/.zshrc
```

#### 사용 방법
```bash
cd /path/to/your/project
claude --model llama4:scout-q4_K_M

# 프롬프트 입력
> JWT 인증 구현해줘
```
