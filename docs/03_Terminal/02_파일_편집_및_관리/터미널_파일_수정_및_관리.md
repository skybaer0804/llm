# 터미널에서 파일 수정 및 관리

## 1. 파일 이름 바꾸기 (mv)
파일 이름을 변경할 때 사용합니다.

```bash
mv route.py router.py
```

## 2. nano 에디터로 수정하기 (가장 간단)
터미널에서 파일을 열고 바로 타이핑해서 수정할 수 있습니다.

```bash
nano router.py
```

**사용법:**
1.  화살표 키를 이용해 수정할 위치로 이동합니다.
2.  내용을 수정합니다.
3.  저장 후 종료:
    *   `Control + O` (`Write Out`) 누른 후 `Enter` (저장 확인)
    *   `Control + X` (`Exit`)를 눌러 터미널로 돌아옵니다.

## 3. Visual Studio Code로 수정하기
VS Code가 설치되어 있다면 훨씬 편하게 수정할 수 있습니다.

```bash
code router.py
```

창이 뜨면 일반 메모장처럼 수정하고 `Command(⌘) + S` (또는 `Ctrl + S`)로 저장하세요.

## 4. 수정 후 서버 재시작
`router.py`의 내용을 바꿨다면, 현재 실행 중인 터미널 창으로 가서 서버를 재시작해야 변경 사항이 반영됩니다.
1.  `router.py`가 돌아가던 터미널 창에서 `Control + C`를 눌러 서버를 끕니다.
2.  다시 명령어를 입력해 실행합니다.

```bash
python router.py
```
