# 시스템 모니터링 명령어

## 1. 현재 메모리 사용량 (macOS)

```bash
memory_pressure
```

또는

```bash
top -l 1 | grep "PhysMem"
```

예: `PhysMem: 45.3G used (15.6G wired), 18.7G unused`

## 2. Ollama가 사용하는 메모리

```bash
ps aux | grep ollama
```

## 3. 라우터 상태 확인

```bash
curl http://localhost:8000/memory | python -m json.tool
```
