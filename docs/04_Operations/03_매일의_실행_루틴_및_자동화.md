# 매일의 실행 루틴 및 자동화 가이드

맥미니 M4를 AI 서버로 활용하기 위한 데일리 실행 시나리오와 스크립트를 이용한 자동화 방법을 안내합니다.

## 1. 데일리 실행 시나리오 (맥미니 부팅 후)

### 1단계: Ollama 실행
*   보통 맥 부팅 시 자동으로 실행됩니다.
*   메뉴바에 아이콘이 없다면 Ollama 앱을 한 번 클릭하여 실행해 둡니다.

### 2단계: LiteLLM 서버 실행
*   터미널을 열고 검색 기능과 OpenAI 규격을 제공하는 중계 서버를 켭니다.
```bash
cd ~/projects/litellm
source .venv/bin/activate
export SERPER_API_KEY=your_key_here
litellm --config config.yaml
```

### 3단계: 안티그래비티(Anti-gravity) 사용
*   맥북에서 안티그래비티를 켜고 설정된 `local-search-model`을 선택하여 질문을 시작합니다.

---

## 2. "한 방 실행" 자동화 스크립트 (`run_ai.sh`)

매번 명령어를 입력하는 번거로움을 줄이기 위해 실행 파일을 만듭니다.

### 스크립트 생성
`~/projects/litellm/run_ai.sh` 파일을 생성하고 아래 내용을 입력합니다.
```bash
#!/bin/bash
# 1. 프로젝트 폴더 이동
cd ~/projects/litellm

# 2. 가상환경 활성화
source .venv/bin/activate

# 3. API 키 등록 (본인의 키 입력)
export SERPER_API_KEY=e7764aab3cd5041751a5957343710520c355fa03

# 4. LiteLLM 서버 실행
litellm --config config.yaml
```

### 권한 부여
터미널에서 한 번만 실행하여 실행 권한을 줍니다.
```bash
chmod +x ~/projects/litellm/run_ai.sh
```

### 실행 방법
이제부터는 터미널에서 `./run_ai.sh` 한 줄만 입력하면 모든 과정이 시작됩니다.

---

## 3. 편의성을 높여주는 도구 활용

*   **Raycast (추천)**: 위에서 만든 `run_ai.sh` 스크립트를 Raycast에 등록하면 단축키 하나로 서버를 켤 수 있습니다.
*   **Stats**: 메뉴바에서 M4 GPU/CPU 사용량을 실시간 확인하여 LLM이 정상 작동하는지 모니터링합니다.
*   **Tailscale (iOS/macOS)**: iPhone에서도 동일한 IP로 맥미니 AI에 접속할 수 있게 해줍니다.
